# Roadmap

이 프로젝트는 기능을 한 번에 밀어 넣기보다, 작은 단위의 이슈와 PR로 나누어 안정적으로 확장한다.

## Phase 1. Foundation

- 프로젝트 구조 정리
- Discord client bootstrap
- slash command 등록 흐름
- TempVoice control panel UI

## Phase 2. Channel Controls

- rename modal
- user limit control
- lock / unlock
- privacy toggle
- waiting room toggle

## Phase 3. Member Controls

- invite
- kick
- trust
- untrust
- block
- unblock

## Phase 4. Ownership

- owner validation
- ownership transfer
- safe delete flow
- stale channel cleanup
