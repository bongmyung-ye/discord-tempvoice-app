# Discord TempVoice App

A Discord bot project for managing temporary voice channels with an interactive control panel.

The goal of this project is to provide a clean TempVoice experience where users can manage their own temporary voice channels through buttons, modals, and slash commands.

## Features

- TempVoice control panel embed
- Channel name and user limit controls
- Privacy, lock, and waiting room options
- Member trust, block, invite, and kick actions
- Ownership claim and transfer flow
- Safe temporary channel deletion
- Slash command based setup

## Tech Stack

- TypeScript
- discord.js v14
- Node.js
- dotenv
- zod

## Getting Started

Install dependencies:

```bash
npm install
```
